## Description

[Joey-learning-nestjs](https://github.com/sanfenggege/Joey-learning-nestjs) framework TypeScript starter repository. Just use to start learning nestjs by Joey Zhang.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
